# Tridivya Wellness

Monorepo containing separate `frontend/` and `backend/` folders.

- `frontend/` – Next.js app
- `backend/` – Express + TypeScript API

To run backend:
1. cd backend
2. npm install
3. cp .env.example .env
4. npm run dev

To run frontend:
1. cd frontend
2. npm install
3. npm run dev
